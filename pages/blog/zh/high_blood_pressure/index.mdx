## 谁受到高血压的影响？

大约三分之一（超过 1 亿）的美国成年人患有高血压。 但这些人中只有一半的病情得到控制。 许多人在 30 多岁或 40 岁出头时会患上高血压，而且随着年龄的增长，它会更频繁地发生。 然而，由于肥胖的流行，越来越多的儿童也患上了高血压。

## 什么是高血压？

血压被定义为当心脏泵血时血液推向动脉壁的力。 高血压 - 也称为高血压 - 是一种血液在高于正常压力下流经血管（动脉）的疾病。

血压是用两个数字来测量的。 第一个或最高的数字是心脏跳动时血管中的压力，称为收缩压。 收缩压是这两个数字中的较高者。 第二个或底部的数字测量您的动脉中血液的力量，而您的心脏在两次跳动之间是放松的。 底部数字是两者中较低的数字，称为舒张压。

正常压力为 120/80 或更低。 如果读数为 130/80，则认为您的血压偏高（第 1 阶段）。 2 期高血压为 140/90 或更高。 如果您多次获得 180/110 或更高的血压读数，请立即就医。 如此高的读数被认为是“高血压危机”。

120/80 和 129/89 之间的读数被认为是高血压前期。 高血压前期患者的血压没有达到应有的低水平，但尚未被认为患有高血压。

## 为什么知道自己是否患有高血压很重要？

早期发现高血压非常重要。 通常被称为“沉默的杀手”，因为它可能没有任何症状，高血压会使您患心脏病、心力衰竭和中风等的风险增加。 根据疾病控制和预防中心的数据，2013 年，美国有超过 360,000 人死亡，其中高血压是主要原因或促成原因。

## 你怎么能控制你的高血压？

高血压的治疗通常从改变生活方式开始，包括减少饮食中的盐分、必要时减轻体重、戒烟、减少饮酒和定期锻炼。

除了改变生活方式外，药物还经常用于降低血压。有几种治疗高血压的药物，每种药物都有益处和风险，您和您的医疗保健提供者应仔细权衡。大多数人服用不止一种药物以将血压降至治疗目标。

您的血压药物应该在几天内开始起作用。然而，由于高血压是一种长期存在的疾病，通常很少或没有症状，因此记住服药可能是一个挑战。组合药物、长效药物或一天一次的药物，可用于减轻服用多种药物的负担并帮助确保定期用药。一旦开始，应该使用药物，直到您的医生告诉您停止。


| 高血压药物的种类 | 行动机制 |
| ------------------------------------- | ------------------- |
| 血管紧张素转换酶抑制剂 | 减少某些使血管收紧的化学物质，使血液流动更顺畅，心脏可以更有效地泵血。|
| β受体阻滞剂 | 放松血管并减慢心率以改善血液流动并降低血压。 |
| 钙通道阻滞剂 | 放松血管，这样心脏就不必那么辛苦地泵血了。 |
| 外周作用α-肾上腺素受体阻滞剂 | 放松血管，使血液更容易流过身体。 |
| 血管紧张素 II 受体阻滞剂 | 阻断某些收紧血管的天然物质的作用，使血液更顺畅地流动，心脏更有效地泵送。 |
| 血管扩张剂 | 放松血管，使血液更容易流过身体。 |
| 中枢作用的α肾上腺素能药 | 降低心率并放松血管，使血液更容易流过身体。 |
| 肾素抑制 | 减少某些会收紧血管的天然化学物质，使血管放松，心脏可以更有效地泵血。 |
| 利尿剂（有时称为“水丸”） | 在肾脏中起作用，以将体内不需要的水和盐分到尿液中。 |
| 联合用药 | 将两种（或更多）不同种类的经批准的血压药物组合成单剂量。 |

控制血压应该是健康生活计划和终身任务的一部分。 高血压对内脏造成的损害在造成严重损害之前不会引起任何症状。

通过掌控自己的健康，您可以帮助控制无声杀手。 这一切都是为了知道谁，什么，为什么，以及如何！
